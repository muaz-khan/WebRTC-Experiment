RTCMultiConnection API Reference / "isInitiator"

What is "isInitiator"?

"isInitiator" is a Boolean property that can be used to detect room owner.

How to detect room owner? 


if(connection.isInitiator) {
    connection.close(); // close entire session
}

Can I set/get this property? 


// yep, you can "set" this property
connection.isInitiator = true;

// you can "get" it as well:
if(connection.isInitiator == false) {
    connection.join({
        userid: 'room-owner-id',
        extra: {},
        sessionid: 'room-id',
        session: {
            audio: true,
            video: true
        }
    });
}
